Relative font size

In HTML , relative font size means specifying how many sizes larger or smaller than the present font size should be. We specify it in the given syntax: <font size = "+n"> or <font size = "-n">

Example

Font size = "-1"
Font size = "+1"
Font size = "+2"
Font size = "+3"
Font size = "+4"
Font size = "+5"
